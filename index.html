Herramientas adicionales

A cotizar durante el primer mes — depende del testeo de cada herramienta.

01 / Infraestructura
Servidor Don Web

VPS donde vive todo el backend: Postgres + PostGIS, n8n, API. Reemplaza los JSON estáticos por una base viva auto-actualizada.

A cotizar con FQC
02 / Vigencia
Google Place API

Validación de business_status de cada POI. Cierra el flanco de “¿este local sigue abierto?” con la fuente más autoritativa del mercado.

Pendiente de testeo
03 / Web Intelligence
SEO

DataForSEO (SERP, backlinks) + monitoreo de citas en IA (Claude, Perplexity) para medir visibilidad digital por marca de la GAF.

Pendiente de testeo
04 / Tráfico
Tracking

Capa de tráfico vial (HERE / TomTom) y footfall opcional por celulares (Foursquare / Cuebiq) para clientes Enterprise.

Pendiente de testeo
Cada herramienta entra al stack solo después de evaluarse en la Semana 2 del Mes 1 y aprobarse en la Semana 3 del Mes 2.
